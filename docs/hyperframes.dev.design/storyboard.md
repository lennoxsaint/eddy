# storyboard.md

`storyboard.md` is the beat-by-beat static contract before animation.

Each frame must include:

- timestamp range
- spoken beat it supports
- static visual composition
- safe zones
- transition into and out of the frame
- proof surface to inspect before animation

If the still storyboard is weak, cluttered, generic, or collides with the screen/camera/captions,
the animated render is blocked. Fix the storyboard first.
